# NimDrop - Nim Shellcode Stager Builder for Termux

A powerful tool for creating Windows shellcode stagers using Nim, designed specifically for Termux.

## Features

- Cross-compile Windows executables from Termux
- Generate Nim-based stagers for shellcode execution
- Built-in Sliver C2 integration
- Simple command-line interface

## Installation

```bash
git clone <https://github.com/invadel/nimdrop>
cd nimdrop
chmod +x install.sh
./install.sh
